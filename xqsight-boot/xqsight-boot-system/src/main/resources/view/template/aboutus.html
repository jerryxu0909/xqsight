我们是易居的技术动力与坚强后盾

易居创新研发中心的服务涉及易居中国各个业务板块，为易居中国的创新思路与创新产品提供切实可行的服务与技术保障。在多年的探索沉淀后形成了一条“房地产+N+互联网”的道路。 易居的能量源既有人才的培养、行业的研究，也有技术产品的研发。作为易居创新研发中心而言，我们的目标与愿景不仅是为易居做互联网技术的服务与创新，更希望把我们的技术、产品模块化，沉淀化，让我的产品、技术走向市场，不仅成为易居的能量源，还会成为一个强大的载体为其他互联网公司提供技术服务。
